# FinSafe - Misleading Financial Content Detection

## Overview
FinSafe detects misleading financial content using NLP and SEBI guidelines.

## Features
- Accepts **text** or **.wav audio** input
- Converts audio to text using Google Speech Recognition
- Sentiment analysis using TextBlob
- Keyword matching for SEBI-related risky claims
- Outputs risk level: **Low**, **Medium**, or **High**

## Technologies Used
- Python
- Streamlit
- SpeechRecognition
- TextBlob
